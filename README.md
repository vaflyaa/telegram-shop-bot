## How to use:

### 1. Setup virtual environment
```
python3 -m venv .venv
source .venv/bin/activate
```
### 2. Install packages
```
pip3 install -r requirements.txt
```

### 3. Input.py
Add your bot's token and admin's id to input.py

### 4. Run bot
```
python3 main.py
```
